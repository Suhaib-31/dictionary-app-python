# Dictionary App 📖

## Overview

Dictionary App is a Python-based application that retrieves word meanings and definitions using a free dictionary API.

Users can search for English words and view their meanings, parts of speech, and example usages.

---

## Features

- Search Any English Word
- View Definitions
- Example Sentences
- Part of Speech Information
- API Integration

---

## Technologies Used

- Python 3
- Requests Library
- REST API
- JSON Parsing

---

## Installation

```bash
pip install requests
```

---

## Run

```bash
python main.py
```

---

## Example

```text
Enter a word: computer

===== Word Details =====

Word: computer

Part of Speech: noun

Definition:
An electronic device for storing and processing data.

Example:
I use a computer for programming.
```
